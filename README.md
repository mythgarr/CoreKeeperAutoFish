# CoreKeeperAutoFish
CoreKeeper AutoFish mod

地心护核者 自动钓鱼Mod
在钓鱼时会喊出鱼的名字并自动和鱼进行拉扯。
支持自动抛竿，抛竿时也会喊出口号。
开始钓鱼后，缩小游戏也可以继续钓鱼。
